# 因子

## ループ
- ライスシャワー = ゴールドシップ + ビワハヤヒデ
- ゴールドシップ = マヤノトップガン + スーパークリーク
- ビワハヤヒデ = スーパークリーク + ゴールドシップ
- マヤノトップガン = ビワハヤヒデ + ライスシャワー
- スーパークリーク = ライスシャワー + マヤノトップガン

- 長距離
- スタミナ

## 有ると嬉しい因子
